About webpage "A web page is a specific collection of information provided by a website and displayed to a user in a web browser."or in other words " A webpage is a document,commonly written in html, that is viewed by a internet browser.a web page can be accessed by entering a url adress into a browser. a web page may contain text, graphics, and hyperlinks to other web pages and files."

Main components of website
  • HTML: HyperText Markup Language(HTML) is the standard markup lanuage for creating web pages to be displayed in a web browser.It can be assisted by technologies such as CSS and scripting languages such as JavaScript.
  • CSS: Cascading Style Sheets(CSS) describes how HTML elements are to be displayed on screen,paper, or in other media.
  • JAVASCIPT: JavaScript often abbreviated as JS, is a programming languagethat conforms to the ECMA Script specification. JS enables interactive web pages and is an essential part of web application.
